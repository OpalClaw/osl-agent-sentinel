# Examples

| File | Description |
| --- | --- |
| `embedded_sdk.py` | Embed sentinel inside an agent runtime via `sentinel.client.Sentinel`. |
| `mcp_gateway.py` | Gate an MCP-style tool runtime — every tool invocation is intercepted. |

All examples run against the default in-memory configuration. Set
`SENTINEL_API_TOKEN` if you want to talk to a remote control plane.
